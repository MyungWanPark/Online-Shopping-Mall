# Online-Shopping-Mall(Full Stack E-commerce)

React, Node.js, Express, MySQL을 활용해 **풀스택으로 구현한 온라인 쇼핑몰** 입니다. <br/>
실시간 상품 검색, 장바구니, 페이지네이션, 상품 추가, 데이터 시각화 등을 구현하였습니다. <br/>
<br/>

✔ 사용자 인증 – OAuth2를 활용한 소셜 로그인(카카오), JWT를 이용한 회원가입/로그인
